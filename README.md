# problem_komiwojazera

Program pozwala rozwiązać problem komiwojazera, który polega na znalezieniu optymalnego, zamkniętego cyklu w zadanym, pełnym grafie, który jest reprezentowany 
przez macierz kosztów.

Kod powstał w ramach zajęć - Zaawansowane progrmowanie obiektowe - na kierunku Automatyka i Robotyka (EAIIB)
